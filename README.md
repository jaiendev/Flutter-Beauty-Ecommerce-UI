## Getting Started

To get started with Bosue UI, follow the instructions below.

### Prerequisites
- Flutter

### Installation
1. Clone the repository: `git clone https://github.com/jaiendev/Bosue.git`
2. Install dependencies: `flutter pub get`
3. Run the app: `flutter run`
